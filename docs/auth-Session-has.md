## Description
Check to see if a value exists

## Synopsis
```php
has(string $name): bool
```

## Parameters
| Name | Description |
| :--- | :---------- |
| name | Name of the value |

## Return
This will return `FALSE` if the value does not exist or `TRUE` if it does.
